# EOD report อิง paid_at ในเขตเวลา Asia/Bangkok

แผนเดิม Bill มีแค่ created_at ไม่มี paid_at ทำให้รายงานยอดขายประจำวันไม่รู้จะอิงเวลาไหน และบิลที่เปิดก่อน
เที่ยงคืนแต่จ่ายหลังเที่ยงคืนจะตกวันผิด

**ตัดสินใจ:** เพิ่ม `paid_at` (timestamp) ลง Bill เซ็ตตอน checkout EOD report รวมยอดจากบิลที่ status = paid
โดย group ตาม `paid_at` แปลงเป็นเขตเวลา Asia/Bangkok นิยาม "วัน" = วันตามปฏิทิน 00:00–24:00 เวลาไทย

**เหตุผล:** ยอดขายต้องตรงกับเงินที่รับจริงในวันนั้น ไม่ใช่เวลาที่เปิดโต๊ะ และร้านอยู่ประเทศไทยจึง fix timezone
ไว้ที่ Asia/Bangkok (ไม่พึ่ง timezone ของ server)
